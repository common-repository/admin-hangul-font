== Description ==

한글 Wordpress를 사용할때 Admin 화면에서 한글폰트가 약간씩 깨져보여서 나눔고딕과 맑은 고딕을 적용시켜 주게 됩니다. 조금 더 깔금한 폰트를 보실 수 있습니다.

= Features =

한글 Wordpress에서 한글로 되어 있는 Admin 화면을 조금 더 깔끔하게 보여주게 됩니다. 향후에 폰트 설정 기능도 추가 예정 입니다.
